05 · FahMai Data-Agent 🥇 Winner

The Finale · AIAT × BOL × NCSA

A secure enterprise AI agent that answers business questions over a company's own data — both structured (SQL) and unstructured (documents) — grounding every answer in OCR-extracted records like bank statements and receipts, with prompt-injection guardrails at every stage. Built with Team Machima; took 1st place.

How it works · 3 pipelines: (1) Agent (ReAct) — Qwen 3 + Typhoon 2-7B Thai LLM routes tools over DuckDB (SQL) + ChromaDB (RAG); (2) OCR-to-data — MinerU turns a statement image into JSON rows in DuckDB; (3) Guardrail — input → reasoning → output checks.
